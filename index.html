WebGL

  • 웹 브라우저에서 제공하는 3차원 그래픽 자바스크립트 API
  • 플래시, 실버라이트, 유니티 등의 플러그인이 필요 없음
  • GPU 가속을 사용한 높은 성능의 그래픽 구현
  • OpenGL ES 2.0 기반
